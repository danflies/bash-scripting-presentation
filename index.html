Command Line Interface (cli)

Are you familar/comfortable with
moving around the command line?

Commands are simple programs that
usually only have one job:

  • ls - List Files and Directories
  • cd - Change Directory
  • mkdir - Make a New Directory
  • rm - Remove a File
  • rmdir - Remove an Empty Directory

Terminology


$ command -option argument ...

  • command/program
  • option/flag/switch
  • argument
I use ls all the time.

However the output doesn't look like a list.

Try
ls -1

It's a one (1).

My ls options


                alias ll="ls -lhGF"
                alias la="ls -lahGF"

My ls Options

  • a - All files including hidden files (files starting with a dot '.')
  • l - Long format
  • h - Human readable file size
  • G - Show group owner
  • F - Append file indicator to name, e.g. * to executables and / to directories

Let's go to your .bashrc file to make some additions.


      $ cd ~
      $ vi .bashrc

You can use another editor besides vi for this file.

Load the file in to your current evironment


                $ source .bashrc

We have to do this now because we've already loaded the shell once and ran the file. We need to tell it manually that we've changed it. After this it will load for each new terminal.
